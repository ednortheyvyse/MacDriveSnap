# MacDriveSnap 26.7.2

MacDriveSnap 26.7.2 focuses on shared catalogue workflows, faster report browsing, and a cleaner catalogue interface.

## What's New

- Added support for choosing a custom catalogue storage folder, including shared network locations.
- Added a Move Catalogue action so an existing local catalogue can be moved to another folder or network share.
- Added a Reveal Folder action beside the catalogue folder chooser in Settings.
- Snapshots now record the Mac name that generated them, making shared catalogues easier to audit across multiple computers.
- Report rows and report details now show the source Mac where available.

## Improvements

- Improved report switching performance by caching report availability and preloading nearby reports.
- Simplified the catalogue view by removing the compact summary metrics and timeline above the report list.
- Moved the Buy Me a Coffee link out of the report detail area so it no longer covers catalogue information.
- Refined the toolbar labels, tooltips, and placement for New Snap, Import, and Compare actions.

## Fixes

- Removed an unsupported SF Symbol that could cause repeated console errors on some systems.
- Avoided the preview path that could fail to load snapshot content in sandboxed WebKit preview processes.
- Cleaned up duplicate MacDriveSnap title/header presentation in the catalogue view.

## Updating

This release supports Sparkle automatic updates from older public versions. You can also download the latest DMG from the GitHub Releases page.
